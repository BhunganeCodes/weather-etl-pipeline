# Weather Data ETL Pipeline

A beginner-friendly data engineering project built with Python.

## Features

- Extract live weather data from Open-Meteo API
- Transform JSON data using Pandas
- Load data into SQLite
- Visualize trends using Matplotlib
- Automate ingestion using Scheduler

## Technologies

- Python
- Pandas
- SQLite
- Requests
- Matplotlib

## Architecture

API
↓
Extract
↓
Transform
↓
Load
↓
SQLite
↓
Analytics Dashboard

## Run

```bash
python main.py
```
